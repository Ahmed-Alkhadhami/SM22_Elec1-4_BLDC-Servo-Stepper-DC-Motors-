# Servo Motor:

This is a simple Servo Motor circuit , which has been controlled via an Arduino. 
The task is done with the help of www.tinkercad.com & www.wokwi.com simulations , also the provided components at SM workshop.

## Repository Contents :
* Circuit Diagram
* Arduino code
* Implementing Video


## Component List :
* Servo Motor
* Arduino UNO
